## Overview

CVE-2023-27372 is an unauthenticated remote code execution vulnerability in SPIP versions before 4.2.1. The vulnerability exists in the password reset form where the `oubli` parameter is susceptible to PHP object injection due to improper deserialization handling.

This exploit provides an interactive shell that:
- Automatically fetches and refreshes anti-CSRF tokens
- Detects vulnerable SPIP installations
- Executes arbitrary commands as the web server user
- Extracts command output from reflected responses

## About This Vulnerability

<Warning>
SPIP versions before 3.2.18, 4.0.10, 4.1.8, and 4.2.1 are vulnerable to this attack. Ensure your SPIP installation is updated to a patched version.
</Warning>
